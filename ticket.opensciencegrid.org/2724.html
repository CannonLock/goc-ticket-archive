    David,

Thanks for the login on your computer. It made it tremendously easier
to track down the problem.

It took quite a while to track down because the original error that
caused the problem is far from the immediate error that caused the
installation to stop.

As best I can tell, you need to have g++ (the C++ compiler that is
part of the gcc compiler suite) installed. I have no idea why it's
really needed, because neither Globus nor PyGlobus need C++. However,
there were early, serious failures due to the lack of C++, so I guess
it is needed.

I can't be sure that this is the only problem, but it is definitely
part of the problem.

Can you install gcc? I believe you need the following RPM:

gcc-c++-4.0.2-8.fc4

You might also need this one, but I'm not sure.

compat-gcc-32-c++-3.2.3-47.fc4

If this doesn't fix the problem, it will at least solve one critical
piece of the problem and will help me continue my search.

Let me know when it's installed, and I'll do more investigation.

-alain
