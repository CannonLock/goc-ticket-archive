Gridcat client (0.7) is returning data for OSG site CIT_ITB_1
(gatekeeper citgrid3.cacr.caltech.edu) which does not match
the information which is displayed with the standard gridcat
web page query for CIT_ITB_1 at:

http://osg-itb.ivdgl.org/index.php?view=sitename&;whichmap=usasiasamerica&numentries=100&sortval=services&id=5

In particular gridcat client queries against CIT_ITB_1 return,
the site name is repeated twice,
two host_names are returned (only one of which matches the standard web page query),
two appdirs are returned (only one of which matches the standard web page query),
two datadirs are returned (only one of which matches the standard web page query),
two tmpdirs are returned (only one of which matches the standard web page query),
two wntmpdirs are returned (only one of which matches the standard web page query).
