    Dear Keith Chadwick,

The OSG Production Team has identified your resource (  FNAL_DZEROOSG_2 - d0cabosg2.fnal.gov ) as running a version of the OSG Software Stack based on a Pacman installation. As of May 31st 2013, OSG will no longer package or release updates to packages which utilize the Pacman installation software. This includes all versions of OSG 1.2.x. To update to a new release please see the installation instructions at https://www.opensciencegrid.org/bin/view/Documentation/Release3/. If there is a technical reason you have not or can not update, please let the Production and Software Teams know via this ticket, and we will work with you to help your transition. We are dedicated to making this transition as smoothly as possible for all OSG resources.

Please respond to this ticket as soon as possible to let us know the planned timeframe for your update and if there is any way we can help you during the transition.

Thank you for your assistance in moving to the newest OSG software stack.
