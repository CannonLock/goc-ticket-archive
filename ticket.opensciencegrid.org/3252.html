    Doug,

I believe I've found the problem. You need to edit your entry in the grid-mapfile so that it is
surrounded by double quotes, not single quotes. I tested this on my CE with each and when using
single quotes it failed. Make the change let me know the results.

[root@atlhn01 osgce]# cat /etc/grid-security/grid-mapfile
'/DC=org/DC=doegrids/OU=People/CN=Douglas Benjamin 438916/CN=1977665841' usatlas1

CHANGE TO:

[root@atlhn01 osgce]# cat /etc/grid-security/grid-mapfile
"/DC=org/DC=doegrids/OU=People/CN=Douglas Benjamin 438916/CN=1977665841" usatlas1
