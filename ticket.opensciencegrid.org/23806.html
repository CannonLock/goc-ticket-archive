    Hi,

If you want the site to appear in the SAM3 dashboards, it should be added to the experiments' topology. This is different per experiment:

o) CMS: add the site to SiteDB
o) ATLAS: add the site to Agis
o) ALICE: add the site to the AliEn LDAP
o) LHCb: add the site to DIRAC

Cheers,
pablo
