    Jan 21, 2011 09:42 PM UTC by OSG-GOC 
    [by Neha Sharma]

Neha,

The TWiki has used X509-based authentication since November 23rd.

That being said, there has also been an issue with Fermilab certificates using a strange DN setup that includes a colon.  I show you using the following DN for OIM access:

/DC=gov/DC=fnal/O=Fermilab/OU=People/CN=Neha Sharma/CN=UID:neha

As this is a FNAL certificate, it unfortunately won't work for authentication with the twiki currently.  We are working on a plan to fix this, but we might have to completely write a custom version of mod_ssl that, in conjunction with FakeBasicAuth, can handle these DNs without question.

In the meantime, if you have a DOEGrids Certificate, you can give us that DN and we can add it to your OIM profile and you can use that certificate to authenticate to the TWiki.

Sorry about any inconvenience this might have caused.  If you don't have a DOEGrids cert, you can request one.
