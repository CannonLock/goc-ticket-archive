I want to add a comment: with the glite-ce it is not possible to submit directly to
the CE using globus-job-run. The only way to submit directly to a glite-CE is through
condor-c itself. In a default installation of a glite-CE the only globus-job-manager
available is the fork one (used by condor-c).
