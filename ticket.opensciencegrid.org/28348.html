    Feb 3, 2016 09:39 AM UTC by OSG-GOC 
    Hello site admin,

The CRAB team plans to use the PhEDEx nodename to direct output. They are picking up the nodename from the stage-out section in the
site-local-config.xml file. Each site has a site-local-config.xml file and it is maintained in the GIT repository at https://git.cern.ch/web/siteconf.git/tree. If your site uses site-local-config.xml from CVMFS, then any update will appear there automatically after a few hours. (If not, please open a ticket.) If you have a local copy of the file on your worker nodes, you would need to update it.
As for the site-local-config.xml file itself. The file has a stage-out within the XML tags "<local-stage-out>". In that section, the stage out command, SE, and PhEDEx nodename need to be defined.
So, in your file, command is missing.
