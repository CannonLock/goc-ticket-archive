[Duplicate message snipped]
    Sep 3, 2014 08:05 PM UTC by Soichi Hayashi 
    I think GOC is the only group that can do this properly. If we ask
VO/project admins, then we will run into the chicken and egg problem; they
can't update before we update FOS, and we can't update FOS before they
update their VO.

I believe following is workflow.

Step 1) Rename all FOS that are renamed via OIM FOS admin (easy!)
Step 2) For each FOS that are removed, go through all VOs / projects that
currently uses it (FOS admin page shows the list), and remove them from the
VO / projects. Then, remove the FOS itself via OIM FOS admin.
Step 3) For each FOS that are merged to another FOS, go through all VOs /
projects that currently uses it (FOS admin page shows the list), and
replace them with the new FOS that is merged to, Then, remove the FOS
itself via OIM FOS admin.

Soichi

Chander has tasked me with 'cleaning up' the field-of-science categories in the OIM project definitions, since many of them are overlaps/duplicates or overly-specific.  Together we've come up with the following groupings. 

Please combine the following fields into one common field called "Ecological and Environmental Sciences":
Ecology
Environmental Biology

Please combine the following fields into one common field called "Medical Sciences":
Biomedical Research
Medicine
Medical Imaging

Please combine the following fields into one common field called "Plant Biology":
Plant Biology
Botany

Please combine the following fields into one common field called "Molecular and Structural Biology":
Molecular Biosciences
Molecular Biology
Structural Biology

Please combine the following fields into one common field called "Evolutionary Sciences":
Evolutionary Biology
Cross-disciplinary Evolution 

Please combine the following fields into one common field called "Physics and Astronomy":
Physics and Astronomy
Physics
Astronomy

Please combine the following fields into one common field called "Astrophysics":
Astro-particle Physics
Astrophysics

Please combine the following fields into one common field called "High Energy Physics":

    High Energy Physics
HEP
Physics - Neutrino
Accelerator Physics
USLHC 

Please combine the following fields into one common field called "Materials Science":
Materials Research
Materials Science

Please combine the following fields into one common field called "Mathematical Sciences":
Mathematics
Mathematical Sciences

Please combine the following fields into one common field called "Computer and Information Science and Engineering":
Computer Science
Computer and Information Science and Engineering 

Please combine the following fields into one common field called "Training":
HTC Training
Training

All other categories can be left as they are.  Let me know if you have any questions or there is some conflict in the groupings.
