The OSG installation puts the certificates (including the DOEGrids
certificate) into globus/share/certificates instead of
/etc/grid-security. If you install just the VDT (not the OSG cache) it
asks you which you want: OSG decided to default to keeping it in
$GLOBUS_LOCATION.
