    Sep 12, 2006 05:46 PM UTC by Rob Quick 
    Ilya,

You can confirm a pacman package has been installed successfully by
doing the following:

# pacman -d 0 -l Package-Name

Here is an example with a successful package installation:

[root@feynman osg]# pacman -d 0 -l CA-Certificates
[*] CA-Certificates...

If the package has not installed correctly you will get an error
message saying either it can not find the package (it has not been
installed) or an error (it has failed during installation for some
reason).

Let me know if you have any other questions,
Rob
