    Hi Tapas,

I'm not really sure what the root cause of the failures is --we'll need to look more into it-- but if you do not see the problem using globus-url-copy, could you try the following?

1. yum install globus-gass-copy-progs
2. add the line: GUCPath=/usr/bin/globus-url-copy to /etc/bestman2/conf/srmclient.conf

And see if srm-copy works after that?

Thanks,
Carl
