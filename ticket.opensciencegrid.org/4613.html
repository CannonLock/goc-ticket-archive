Hi,

I have been asked by the OSG Executive to being producing a regular usage report for the OSGEDU VO similar
to the attached report for the OSG VO. The report is most useful when given access to a VOMRS database since
it can group usage by user rather than just by cert; however in the absence of a VOMRS DB the VOMS DB allows
the tracking of users with zero usage during the time in question. It also allows the accounting of usage where
the VO could not be obtained.

The query to be run against the VOMS DB is as follows:
select '' as firstname,
u.cn as lastname,
m.userid as member_id,
u.cn,
g.dn as group_name,
ifnull(r.role, '') as role_name
from m
join usr u on (m.userid = u.userid)
join groups g on (m.gid = g.gid)
left join roles r on (m.rid = r.rid)
order by lastname, cn, group_name, role_name;
