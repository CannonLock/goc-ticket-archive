    Dec 22, 2008 08:48 PM UTC by Karthik Arunachalam 
    Using /opt/osg as the value for 'VDT installation root' worked.

[root@tier2-01 osg]# $VDT_LOCATION/vdt/setup/configure_cemon --vdt-install
/opt/osg --server y --consumer is1.grid.iu.edu --topic OSG_CE --dialect
RAW
The following consumer subscription has been installed:
HOST:    is1.grid.iu.edu
TOPIC:   OSG_CE
DIALECT: RAW

[root@tier2-01 osg]# $VDT_LOCATION/vdt/setup/configure_cemon --vdt-install
/opt/osg --server y --consumer is2.grid.iu.edu --topic OSG_CE --dialect
RAW
The following consumer subscription has been installed:
HOST:    is2.grid.iu.edu
TOPIC:   OSG_CE
DIALECT: RAW

It would have been simple to use $VDT_LOCATION instead of 'VDT
Installation root', which was confusing for me.

-Karthik
