    Hi Jeff,

You're using the wrong version of the globus-gram-job-manager-scripts -- it's globus-gram-job-manager-scripts-4.2-7.el6 according to the profile. Note that it doesn't have "osg" in the version, so it doesn't have the OSG-specific patches. One of our patches is to load osg-job-environment.conf.

You should try to yum update globus-gram-job-manager-scripts, and make sure that the version you get is from osg.
The current production version we ship is globus-gram-job-manager-scripts-4.1-3.1.osg.el6.

By the way, it looks like most of your other globus packages aren't from OSG either -- if you run into strange errors, you should update those as well. Here's a one-liner that might help:
rpm -qa | grep \^globus | grep -v \.osg | xargs yum upgrade
