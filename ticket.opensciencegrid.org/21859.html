    As presented and discussed during the 2 latest WLCG Ops coordination meetings, during August CERN is going to upgrade all CVMFS installation servers to version 2.1. (https://indico.cern.ch/event/321389/contribution/5/material/1/0.txt)

This change requires all sites to upgrade to the latest version of the CVMFS client ( 2.1.19) released on 28th May cause it fixes known issues.

From the Experiment Dashboards it looks like your site is still running an old version of the client < 2.1.19. Could you please plan the upgrade of your installation?

Please note that the upgrade requires only the related rpm upgrade and you can find more details at: http://cernvm.cern.ch/portal/filesystem/cvmfs-2.1.19

Please also  note that in case of CVMFS -NFS installation we cannot determine the version installed at your site, therefore if you have already upgraded please just close the ticket.
