    I figured out an alternative way to do this using srmmkdir on bsrm-3.t2.ucsd.edu, like so:

srmmkdir -2 -delegate=false  srm://bsrm-3.t2.ucsd.edu:8443/srm/v2/server?SFN=/hadoop/sbgrid/user/webportal/workflow/testdir

Then I use lcg-cp to copy data files there:

lcg-cp -b -D srmv2 --vo SBGrid -t 2400 --verbose file:TA1305_F8_test_crysal_friedeltrue_xds_FP_FreeR_flag.mtz srm://bsrm-3.t2.ucsd.edu:8443/srm/v2/server?SFN=/hadoop/sbgrid/user/webportal/workflow/testdir/sampledata.mtz
