greenc@.... sent this Notes To Requester:

Hi Ben,

Sorry it took me a few days to get to this -- I've been on vacation
since last Thursday and just got back this morning.

First, please use the VOMRS rather than the VOMS UI -- this is more
straightforward, we get more information about the various stages of the
process and can help when things go sour.

The URL is OSG VOMRS
<https://voms.opensciencegrid.org:8443/vomrs/osg/vomrs>;, but before you
connect please do the following:

If you are using Firefox or some other browser that doesn't use the
Windows certificate store:

1. Go to your browser's CA certificate list ("Authorities" in
Firefox's Options/Preferences->Advanced->Certificates) and ensure
that there are entries for:
1. ESnet Root CA 1; and
2. DOEGrids CA 1;
2. Check also that they have not expired (select, then click "View"),
and that they are able to to authorize web sites, email users and
software (select, then click, "Edit").
3. If certificates are missing and expired, obtain them from:
http://www.es.net/CA/ESnet-Root-CA-1 and
http://www.doegrids.org/CA, and don't forget to check the
authorization boxes as they are imported. To import them on
firefox, from "Authorities" in Firefox's

    Options/Preferences->Advanced->Certificates click Import and
import them after utilizing "save target as" for the PEM versions
of each certificate.

If you are using the Windows certificate store:

1. Right-click the IE icon on your desktop and select, "properties";
or select, "Internet Options" from the control panel (you may have
to select, "Classic View" in order to find it). Select the,
"Content" tab and hit the, "Certificates" button. The ESnet root
certificate should be under, "Trusted Root Certification
Authorities" and the DOEGrids CA certificate under, "Intermediate
Certification Authorities."
2. If they exist, check the expiration date by selecting and pressing
the, "View" button, and the authorization abilities of each
certificate under the, "Advanced" button.
3. If they are missing, visit the links in step 3 of the Firefox
instructions but this time download the DER versions of each
certificate to the desktop and double-click on them. The Windows
import wizard should put them in the right place.

/Now/, you can click on the link OSG VOMRS
<https://voms.opensciencegrid.org:8443/vomrs/osg/vomrs>; link and attempt
to join the VO. You may have to present your DOE certificate multiple
times -- just be sure you select the same certificate each time. Select
Registration Phase I and follow the instructions. Don't forget to follow
the instructions in the email you'll receive in order to progress your
application.

Reply on this ticket to let me know how it goes.

Thanks,
Chris.
