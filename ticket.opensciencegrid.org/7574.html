Sarah,

I am sorry no such command. So need to run some SQL

1) Check that the linkgroup that spacereservation belongs to allows online:

select onlineallowed from srmlinkgroup where id in (select linkgroupid from srmspace where id=?);

substitute ? with id of space reservation you are talking about if query return 1 you are OK if not,
modify it:

update linkgroup set onlineallowed=1 where id in (select linkgroupid from srmspace where id=?);

2) modify space reservation:

update srmspace set accesslatency=(select id from srmaccesslatency where name='ONLINE') where id=?'

replace ? with id of space reservation in question.


    I will make admin command to allow this in the future.
