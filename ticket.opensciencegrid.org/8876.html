You have to be careful about using comments, and read the RT update
webpage carefully, where it says "Update Type: Comments (Not sent to
requestors)".  In this particular case, the GOC and ultimately GGUS were
the requestors, so RT did exactly what it was supposed to do, record the
comment in RT only and not send it to the GOC and GGUS.

Our own RT documentation also explains the difference:

https://www.racf.bnl.gov/docs/rt/rtexpertuser#rt_respond

So...

Comment - internal, visible to admin staff only, not sent to the user or
other ticket systems.  Use this to make a comment about the ticket or
communicate a message to other admin staff.

Reply - use this to reply to the user that opened the ticket.

~Jason
