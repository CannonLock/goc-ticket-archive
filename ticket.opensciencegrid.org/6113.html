It looks to me like you haven't installed the vo-client package, so
Pacman can't update it. For Pacman "-update" means "take a package
that was already installed, and look for an update." It will not
install a package if it hasn't been installed before.

So I think you should do this:

pacman -get OSG:vo-client

Now I haven't seen everything else you've done up to this point... Did
you install the OSG client from the OSG cache? If you want me to be
more definitive, send me the following file from your installation:

$VDT_LOCATION/o..pacman..o/logs/commands

It seems to me that the OSG client should include this by default, and
it was overlooked. I'll talk to a few people about it and see what we

    can do about it.
