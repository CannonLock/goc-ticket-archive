    Sep 25, 2017 09:01 PM UTC by Matyas Selmeci 
    Hi Trey,

What the probe is designed to do is test submission from condor via grid
universe to a condor-ce.  This means it submits a job to a condor (not
condor-ce!) running on a local machine, which then sends the job over to the
condor-ce.

This means you'll need to have condor running on the machine you are running
rsv from. I haven't verified, but I think a schedd is enough -- use
DAEMON_LIST = SCHEDD, MASTER
in the condor config.

(You should undo the changes made earlier in this ticket.)
