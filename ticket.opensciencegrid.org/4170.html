From the install docs:

"The CEMon (CEMonitor) service is responsible for providing information
coming from the Computing Element (CE) and is being implemented as an
optional component in the OSG Compute Element (CE) architecture in
support of the ReSS (Resource Selection Service) OSG project. This
service is capable of collecting the CE resource information by using
Generic Information Provider (GIP). GIP works with the Condor, PBS, LSF
and SGE job managers."
