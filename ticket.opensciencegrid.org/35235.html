    Nov 20, 2017 08:48 PM UTC 
    Hello Enrico,

I would suggest creating one entry and mapping everyone to a single gm2 user. The DNs you will find from the Fermilab jobs are of the form

/DC=org/DC=cilogon/C=US/O=Fermi National Accelerator Laboratory/OU=People/CN=Full Name/CN=UID:username

So something like /DC=org/DC=cilogon/C=US/O=Fermi National Accelerator Laboratory/OU=People/CN=*/CN=UID:* grabs all of the analysis jobs. The production DN is a little different:

/DC=org/DC=opensciencegrid/O=Open Science Grid/OU=Services/CN=production/gm2gpvm01.fnal.gov

but it is always exactly that. So in the gridmap file you can add those DNs and map them to whatever username you want to use for the g-2 jobs.
