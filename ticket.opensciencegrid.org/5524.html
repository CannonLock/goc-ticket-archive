Hi,

As described in the TWiki installation guide at
http://twiki.org/cgi-bin/view/TWiki/TWikiInstallationGuide it
is important to protect the configure script.

The TWiki security team was made aware of an exploit with
an unprotected configure script:
http://www.milw0rm.com/exploits/6269

Your configure script https://twiki.grid.iu.edu/bin/configure
is not secured. I suggest to use Apache authentication to
require a list of admins and/or IP addresses to access the
configure script.

Regards,
Peter
