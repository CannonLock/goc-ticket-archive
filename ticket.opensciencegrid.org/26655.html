Hello OSG Support,

My name is Sergei Chekanov and I am using OSG resources to perform research in the field of HEP. It would be helpful to me to have the
HEPMC software (version 2.06.08  ) and ProMC (1.5)
available on the grid.  Please
find details on installing this software at the following web site:

HEPMC: http://lcgapp.cern.ch/project/simu/HepMC/
PROMC: https://atlaswww.hep.anl.gov/asc/promc/

Please first install HEPMC. Then, when compiling ProMC, use the HEPMC
instillation, i.e. run the command:

./install.sh [promc_dir] [hepmc_dir]

The installation instruction of ProMC is:

https://atlaswww.hep.anl.gov/asc/wikidoc/doku.php?id=asc:;promc:installation

best, Sergei
