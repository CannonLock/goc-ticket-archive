    Pffft... I dunno.  Maybe /etc/grid-security/vomsdir?  That's where the
LCG wants to find this public certificate.  Do this:

for i in `locate cert.pem`; do openssl x509 -in $i -subject -noout |
grep "fnal.gov" &> /dev/null; if [ $? -eq 0 ]; then echo $i; fi; done

That should find it...
