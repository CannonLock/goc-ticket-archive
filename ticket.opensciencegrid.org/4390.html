On Nov 16, 2007, at 3:20 PM, goc@.... wrote:

> Support Centers,
>
> There is a new VO package which includes the updated entries in the
> vomses file for the CERN VOMS changes. It also includes a new
> format for the gums.template that will allow both versions 1.1 and
> 1.2 of GUMS to function properly.
>
> All OSG resources are encouraged to upgrade.
>
> For resources that have updated to 0.8.0 please use:
>
> pacman -update OSG:vo-package
>
> For resources at OSG 0.6.0 please use:
>
> pacman -update OSG:vo-0.6
>
> For resources still using 0.4.x we are no longer producing packages
> for 0.4 installation please update your OSG middleware to the
> current 0.8.0 stack.
>
> For OSG Client users please use (Note: This not for wn-client and
> works for both 0.6 and 0.8 releases.):
>
> pacman -update OSG:vo-client-0.6
>
> Please contact the GOC with any questions.
