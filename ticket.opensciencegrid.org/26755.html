Subject: Known issue with glideinWMS and HTCondor 8.3.7 and 8.3.8

HTCondor versions 8.3.7 (released on 2015-07-23) and 8.3.8 (released
on 2015-08-27) contain a bug that can affect glideinWMS factories and
possibly glideinWMS VO front-ends. Some HTCondor configurations cause
the shared port daemon to leak file descriptors. The problem is
triggered when some, but not all, of the HTCondor daemons are configured
to use a shared port. Note that the latest OSG Software release (version
3.3.1 of 2015-09-08) contains HTCondor 8.3.8, and so glideinWMS installations
that upgrade to this OSG release are affected.

Further information and a workaround is available in the 3.3.1 release notes:
https://twiki.opensciencegrid.org/bin/view/Documentation/Release3/Release331
