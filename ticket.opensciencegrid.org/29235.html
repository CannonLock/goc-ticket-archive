    Chris,

Thanks for the suggestion.  I did some additional testing using your
suggested command.  It seems that part of the Subject generated by
Globus grid-cert-request is causing the problem.

The subject generated by grid-cert-request looks like this:

/DC=com/DC=DigiCert-Grid/O=Open Science
Grid/OU=Services/CN=host/wolf.xray.aps.anl.gov

If I remove "host/", so it looks like this:

/DC=com/DC=DigiCert-Grid/O=Open Science
Grid/OU=Services/CN=wolf.xray.aps.anl.gov

The CSR gets accepted.  I'm going to modify my CSR script to use the
openssl command directly and submit my CSRs.

With the above work around, this case can be closed.

Thanks for the help.

Roger S.
