    Jun 25, 2008 11:17 PM UTC by CIGI 
    Hi Britta,

I believe this is a permission issue. The vomses file needs to be owned
by root with 644 permissions.

To change ownership, run the following command (As root):
%chown root:root /ldcg/stow_pkgs/ldg-4.5/ldg/vdt/glite/etc/vomses

Then rerun voms-proxy-init

(Also make sure that the /ldcg/stow_pkgs/ldg-4.5/ldg/vdt/glite/etc
directory is also owned by root:root with permission 755.)

Thanks
Anand
