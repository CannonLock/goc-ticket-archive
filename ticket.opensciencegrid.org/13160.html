    Dec 10, 2012 10:02 PM UTC by Anand Padmanabhan 
    The security team noticed that the version of Condor being run at your site (7.6.4) is vulnerable an attack where an attacker could bypass the target site's host-based authentication. More details on the vulnerability can be found at and http://research.cs.wisc.edu/htcondor/security/vulnerabilities/CONDOR-2012-0002.html . We would recommend you update your condor to the latest version (7.8.6) released by OSG that resolves this issue.

Please let us know if you have any questions.

Best
Anand
