    Nov 12, 2008 05:39 PM UTC by Scott Kronenfeld 
    Hi Ian,
One possibility is that RSV is generating some incorrectly named gratia scripts.

Do you have files in this directory?
$VDT_LOCATION/osg-rsv/output/gratia/

There 's a known bug that generates the gratia scripts with an
incorrect name - there is a newline in the filename which causes the
gratia scripts to not be processed correctly.  If this bug is
occurring on your system, you will see files in this directory with a
question mark (or possibly a newline) in the name (my shell displays
the newline as a '?', but yours may differ)

Arvind supplied a bug fix for this, which the VDT will release soon in
a lettered update.

-scot
