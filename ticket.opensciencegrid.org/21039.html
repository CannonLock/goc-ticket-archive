    May 16, 2014 03:07 PM UTC by Tim Cartwright 
    GyeongRyoon:

Your job (test2.sh) writes output to a file contained in a directory (RESULT), and your submit file asks HTCondor-G to transfer back the entire output directory. However, HTCondor-G does not support the transfer of directories, only files, so the transfer fails.

Try changing your script to write a single file in the sandbox, and then specify that file in the transfer_output_files line of your submit file.

— Tim
