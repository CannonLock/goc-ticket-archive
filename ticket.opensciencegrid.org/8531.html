    May 4, 2010 11:09 PM UTC by OSG-GOC 
    Hi Steve,
A host at your site, magic.cse.buffalo.edu, is reporting to an older Gratia server.
Current transactions targeted at "gratia.opensciencegrid.org<http://gratia.opensciencegrid.org/>;" are now being automatically forwarded to "gratia-osg-prod.opensciencegrid.org<http://gratia-osg-prod.opensciencegrid.org/>;". However the forwarding server is going to be retired in approximately four weeks.

As a result OSG is requesting that you update this pointer.

There are two ways to go about this:

1) If that host is an OSG CE, you can upgrade it to the latest version of OSG 1.0 or 1.2, as appropriate. One benefit of the
upgrade is that this address will automatically be updated.

OSG 1.2 update instructions:
https://twiki.grid.iu.edu/bin/view/ReleaseDocumentation/OSG12UpdateInstructions

OSG 1.0 update instructions:
https://twiki.grid.iu.edu/bin/view/ReleaseDocumentation/OSG106UpdateInstructions

If you're using the Gratia RPM, we'll assume that you know how to get upgrades for it.

2) If an upgrade is not possible at this time, you can tweak the Gratia configuration directly to point at the correct host.  Let
us know if you need to do this instead, and we'll give you some one-on-one help. Also, the OSG Campfire is available three times a
week for interactive help: (Marco please provide a link).

Please let us know if you have any questions or concerns, and we'll be glad to help!

Thanks,
Dan Fraser, Marco Mambelli, and a host of dozens at OSG.
