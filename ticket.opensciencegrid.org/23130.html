    Nov 15, 2014 12:02 AM UTC 
    Hi All,

I just build a release in koji for graphtool. Graphtool is the library that handles almost all the connections to the database that Gratia Web requires. There was a semaphore in the connection_manager.py script in the MySqlDatabase class that was getting full when it could not create a connection for any expected or unexpected reason. After the semaphore was full the Gratia Web server became unresponsive, so i fix it so now if for any reason the connection fails the semaphore is being released. I also included some changes to solve GRATIAWEB-60. I already tested this changes in my local machine and in a fermicloud machine. The release number is 21. I still have to look after how to handle timeouts in statements that are being executed and other concerns i got about the database connections issues gratiaweb still have.

Thanks,
