    Jan 21, 2009 08:33 PM UTC by shoover@.... 
    I posted on osg-sites and got the following clean up script from Steve
Timm.

#!/bin/bash
# Source the globus setup
. /usr/local/osg/setup.sh
cd $GLOBUS_LOCATION/tmp
find . -xdev -type f -a \( -atime +7 -o -mtime +14 \) -exec rm -f {} \;

I ran the find part first to verify what the script would get, then
added the rm and cleaned up the directory. I've added it to a weekly
cron and should have all of the non-existent job files cleaned up in
14 days.

Thanks,
Sam
