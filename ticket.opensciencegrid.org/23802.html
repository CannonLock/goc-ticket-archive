    Jan 29, 2015 10:43 PM UTC by Brian Lin 
    Edgar pointed out something to me, it looks like RSV is trying to run the GRAM tests. Make sure you have /etc/osg/config.d/30-rsv.ini configured properly, specifically the following lines:

gram_ce_hosts=...
htcondor_ce_hosts=...

If you're running both GRAM and HTCondor CE's on the same host, they should go in both lines but if not, they should go on the corresponding line. Also, make sure that you have both GRAM/HTCondor CE gateways enabled in /etc/osg/config.d/10-gateway.ini if you have a mix of CE types. After that's done, verify your changes and apply them with:

$ osg-configure -v
$ osg-configure -c
