    I apologize for my slow response. I've been traveling.

It looks like you have the wrong certificate, but you can easily get
it from here:

http://pki1.doegrids.org/displayBySerial?op=displayBySerial&serialNumber=0x79ce

You'll note that there are two different bits of text you can use on
that page. You should use the one labeled:

> The following format can be used to install this certificate into a
> server.
>
> Base 64 encoded certificate

Instead, it looks like you used this one:

> Base 64 encoded certificate with CA certificate chain in pkcs7 format

So take the first one instead, and you'll have the correct host
certificate. Then try the installation again.

Let me know how it goes!
