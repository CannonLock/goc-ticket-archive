    Mar 17, 2011 04:48 PM UTC by OSG-GOC 
    [by Alain Roy]

Hi GOC,

I believe there is an error in the GOC-distributed vomses file. You have:

% grep voms.opensciencegrid.org vomses
"osg" "voms.opensciencegrid.org" "15027" "/DC=org/DC=doegrids/OU=Services/CN=host/voms.opensciencegrid.org" "osg"

Note that the DN has "host" in it. I think it should be "http" instead. That is, it should read:

"osg" "voms.opensciencegrid.org" "15027" "/DC=org/DC=doegrids/OU=Services/CN=http/voms.opensciencegrid.org" "osg"

If you're doubtful about this, check out the following URL. VOMS itself suggests using http instead of host.

https://voms.opensciencegrid.org:8443/voms/osg/webui/config

It's not urgent to update this, but the next time you ship it you should fix it.

Thanks!
-alain
