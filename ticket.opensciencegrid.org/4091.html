    Oct 2, 2007 03:40 PM UTC by Chris Green 
    Hi,

If you're interested in installing the probes on your existing 0.4.0
installation, please see:

https://twiki.grid.iu.edu/twiki/bin/view/Accounting/ProbeInstallation

Note that you may wish to install the RPMs in their default locations
and use a soft link from /opt/vdt/gratia to $VDT_LOCATION/gratia. If you
are using Condor as either your primary batch system or fork job manager
you will need to apply patches to the globus jobmanagers; please see the
instructions and let me know if you have any trouble.

Thanks,
Chris.
