    Feb 20, 2013 12:18 AM UTC by Horst Severini 
    Hi again,

here's another update. Thanks to Burt Holzman, we have OUHEP_OSG's GIP
working again. He indeed confirmed that the old GIP version
that came with osg-1.2 does indeed not like any stderr output,
so that shlib incompatibility did indeed cause the GIP to stop working.

He provided a patch which I applied, and now we're all green again in MyOSG,
so hopefully the ATLAS release validation jobs will start flowing in soon
again as well.

We still see the gratia gratia RSV probe warning, but I think that's
not a critical error, since we're definitely seeing Gratia records
being uploaded.

I'll ask Joel to apply the patch at LUHEP_OSG as well, so we can make sure
that this patch is a general fix for osg-1.2 sites.

Thanks again,
expecially to Burt!
