    Apr 14, 2018 09:08 PM UTC 
    Hi,

Sorry fort the delay. To be precise GLIDEIN_Site is for Users to request the site,  but is not used for GRACC accounting. This one is somewhat arbitrary, we are stuck with whatever we agreed on or otherwise run the risk of users no longer matching their jobs to the site.

On the other hand, GLIDEIN_ResourceName is the factory entry variable that has to match Resource Group name exactly, to get GRACC accounting to work.

Rich, we can see what you have set from your side here:
condor_status -pool collector.opensciencegrid.org:9619 -schedd -const 'name=?="its-condor-ce2.syr.edu"' -l | grep 'OSG_Resource[ G]'
OSG_Resource = "SU-ITS-CE2"
OSG_ResourceGroup = "SU-ITS"

So it looks like for CE2 it should be "SU-ITS"

Jeff Dost
OSG Glidein Factory Operations
