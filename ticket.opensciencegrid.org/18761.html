    As long as both client and server software are up to date, it should work if they are using sha-1 or sha-2 certificates at either end. If either the client or server is using a sha-1 cert, and has outdated software installed, it may/probably won't work with sha-2 certs. I believe the following page still has the update versions required: https://twiki.opensciencegrid.org/bin/view/Documentation/Release3/SHA2Compliance
