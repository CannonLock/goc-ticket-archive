    [John Weigand]

This is related to the Gratia-APEL interface which sends Gratia data to the APEL/EGEE portal for Tier 2 MOU purposes for selected resource/sites.  Currently, this interface uses a manually maintained configuration file as the basis for the resources/sites involved.

In mid-November, we will be implementing a change that will use the MyOSG WLCG Information InteropAccounting flag to identify these resources/sites.  This flag for HEPGRID_UERJ is currently set to False so when this change is implemented, we will no longer be forwarding data for this resource to APEL.

If you wish to continue having this data forwarded, please change the InteropAccounting flag to True.
http://tinyurl.com/2um4at4

The WLCGInformation AccountingName should also be populated.

You may also want to evaluate the WLCGInformation associated with resource HEPGRID_UERJ_OSG64 as well to determine if it is correct.  We are currently not uploading data for that resource.
