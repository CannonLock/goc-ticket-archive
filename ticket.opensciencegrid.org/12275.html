On 06/15/2012 10:14 AM, Open Science Grid FootPrints wrote:
> [Duplicate message snipped]
    Jun 15, 2012 03:14 PM UTC 
    voms.fnal.gov and voms.opensciencegrid.org are independent IP names and addresses:

nslookup voms.fnal.gov
Name:   voms.fnal.gov
Address: 131.225.152.140

nslookup voms.opensciencegrid.org
Name:   voms.opensciencegrid.org
Address: 131.225.152.138

One is not an alias for the other.

Furthermore, at the backend in the VOMS server, the VOMS instances operate using
separate credentials linked to the IP name in order to complete the necessary
SSL handshake.

So bottom line, you cannot attempt to "cross reference" a VO.

-Keith.
