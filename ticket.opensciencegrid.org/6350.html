Hi Prakashan,
The problem is that you have too old of a vdt-update-certs program.
In VDT 1.10.1i we released an update which allowed users to fetch
distributions other than the VDT convenience distribution.  It is not
sufficient to just set cacerts_url, you need to have the newer version
of vdt-update-certs as well.

We have a large update to VDT 1.10.1 coming out this week, which will
include an automatic updater script.  I recommend waiting until this
is released and using this.   It will update your vdt-update-certs
program to a newer version, and then you will be able to use
cacerts_url to fetch the GOC's distribution.  Watch for emails from
the goc about this release.

Let me know if you have further questions.

Thanks,
