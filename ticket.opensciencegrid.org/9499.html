    [John Weigand]

This is related to the Gratia-APEL interface which sends Gratia data to the APEL/EGEE portal for Tier 2 MOU purposes for selected resource/sites.  Currently, this interface uses a manually maintained configuration file as the basis for the resources/sites involved.

In mid-November, we will be implementing a change that will use the MyOSG WLCG Information InteropAccounting flag to identify these resources/sites.  This flag for AGLT2_CE_2 is currently set to True.

However, we are NOT currently sending data to APEL/WLCG for AGLT2_CE_2.  We are only sending data for AGLT2.  In reviewing Gratia data, the AGLT2 is a very small percentage of the total for both resources/sites so this oversight may not have been obvious.  It has been proportionately the same for all 2010.

| AGLT2_CE_2 | usatlas | 2010-10 |      15145 |       3 |     150 |

If this setting is correct, please confirm. If incorrect, please change to False.
