    Apr 30, 2015 03:15 PM UTC 
    Hi,

The behavior of autofs described in this ticket are normal. Only mounted filesystems will be seen when listing the contents of the /cvmfs/ directory. The /cvmfs/nova.opensciencegrid.org file system will only be visible if it has been accessed. This is transparent to any programs/processes accessing files. All systems are configured with this file system, and our monitoring system has not reported any failures to mount the file system. Let me know if there are any questions or concerns.
