The OSG Pacman cache does not contain a Bestman package, but you can get it from the VDT directly. For OSG,
the corresponding VDT version is 1.10.1, so the command to install Bestman is:

pacman -get http://vdt.cs.wisc.edu/vdt_1101_cache:;Bestman

It does rely on GridFTP, but this installation will bring along GridFTP. It does not install edg-mkgridmap, but you
can install that separately.

pacman -get http://vdt.cs.wisc.edu/vdt_1101_cache:;EDG-Make-Gridmap

-alain
