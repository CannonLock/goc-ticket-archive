    Apr 23, 2013 12:23 AM UTC by Matyas Selmeci 
    Hi,

Steve Timm responded and told me that the metric probe is run from the gratia-consumer in RSV. So you should do the following:
1. Make sure "rsv-consumers" is installed
2. Run "rsv-control --enable --host <YOUR RSV HOST> gratia-consumer"
3. Run "rsv-control --on --host <YOUR RSV HOST> gratia-consumer"

trying to reverse engineer it.
