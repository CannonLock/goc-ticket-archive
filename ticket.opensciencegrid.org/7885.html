    Hi,

atlas.bu.edu is still contacting gratia.opensciencegrid.org
instead of gratia-osg-prod.opensciencegrid.org.

This means that they have an old version of the Gratia Probes installed
and they need to solve the problem one of the following ways:

1. Use pacman update (see
https://twiki.grid.iu.edu/bin/view/ReleaseDocumentation/UpdateInstructions)
to ensure you are running the latest version of the probes for your OSG
version.

2. As a last resort, manually edit the ProbeConfig file and change
SOAPHost and/or CollectorHost to the correct hostname and port 80
(please note, the port has changed to 80!)
