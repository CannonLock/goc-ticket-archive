    Nov 18, 2009 09:11 PM UTC by Arvind Gopu 
    Suchandra requested we update the OSG-1.2 cache's VO-package and VO-client .pacman files with files he provided. The files are copied and pasted below:

vo-package.pacman
#
#
description ('OSG VO Package v25')
url ('http://www.opensciencegrid.org/osg/';)
platformGE('*')
package ('http://vdt.cs.wisc.edu/vdt_200_cache:;VDT-Core' )
download('http://software.grid.iu.edu/osg-1.2/tarballs/vo-package-25.tar.gz';)
shell('vdt-begin-install vo-package');
shell('vdt-untar vo-package-25.tar.gz');
shell('vdt-end-install');
uninstallShell('vdt/sbin/vdt-uninstall vo-package');

{
exists('monitoring')
OR
softLink('osg/etc', 'monitoring')
}

vo-client.pacman
#
#
description ('OSG VO Client v25')
url ('http://www.opensciencegrid.org/osg/';)
package ('http://vdt.cs.wisc.edu/vdt_200_cache:;VDT-Core' )
download('http://software.grid.iu.edu/osg-1.2/tarballs/vo-client-25.tar.gz';)
shell('vdt-begin-install vo-client');
shell('vdt-untar vo-client-25.tar.gz');
shell('vdt-end-install');
uninstallShell('vdt/sbin/vdt-uninstall vo-client');

I updated the vo-package's trunk with these files (replacing _1.2 files); then I re-released version 25 of the VO-package, and re-installed the production osg-1.2 cache. The change is reflected in the vo-*.pacman files in http://software.grid.iu.edu/osg-1.2/
