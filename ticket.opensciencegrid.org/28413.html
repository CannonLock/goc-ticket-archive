    Hi Chris,

I have been working on this issue this morning. I think I have found the cause and fixed theproblem.

Around January,  our team updated the PBS installation on the osg-ce node with the new version. Because osg-ce is configured as PBS client node, the update removed the directory /var/spool/PBS/server_priv/accounting which holds the accounting data. As a result, the gratia probe does not find any job record and reports nothing. I just manually created this folder and have re-synchronize the job log. We can check a few hours later to see the problem will be fixed.
