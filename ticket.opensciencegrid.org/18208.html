by /DC=com/DC=DigiCert-Grid/O=Open Science Grid/OU=People/CN=Joe Boyd 579
    Dec 12, 2013 06:03 PM UTC 
    Hi Shaun,

This sounds a lot like a SEG issue.  Can you check to see whether it is running?  If it is, is it producing an updated file in this directory:

/var/lib/globus/globus-seg-pbs

?

If no - try restarting the SEG and see if it recovers.  This sometimes struggles with log rotation by PBS; see if this issue recurs (if it does, we have some tricks).

If yes - try killing the globus-job-manager processes on the host ("killall globus-job-manager" or "killall -u nova globus-job-manager").  Does that cause the state to recover?  "service globus-gatekeeper restart" doesn't tend to help the CE from getting "un-stuck" as it doesn't kill off the persistent processes associated with the user jobs.

Brian
