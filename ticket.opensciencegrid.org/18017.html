    Dec 9, 2013 05:29 PM UTC 
    Hi Soichi,

From a grid perspective, Crane is as production as it's likely to get.  Like all other Nebraska resources, we provide opportunistic and preemptable resources for the grid.  Right now the UN users have not filled out the cluster, therefore there is a lot of room for grid jobs.  But that means that when a UN user does run a large job, it will preempt a lot of grid jobs.  The preemption rate is probably pretty low, but the number of jobs preempted at once is probably really high.

Soichi - unless you have significant issues running at Crane, I would recommend you continue to run there.  The OSG VO is receiving thousands of cores continuously on the cluster.

-Derek
