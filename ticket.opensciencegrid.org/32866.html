    Ah ok, so Chrome uses the keychain on Macs.  So open up your keychain,  import or if it's already been done then search for CILogon OSG CA 1 cert.
Highlight and right click the cert and select get info.  A pop up box will come up and you'll then drill down on trust.  Now select When using this
certificate, drill down the menu and select always trust.  That should take care of it for you.
