SRM Client 1.25 Available for OSG 0.4.x - (OSG-GOC Ticket # 2785)

Fri, 5 Jan 2007 0:04:55 UTC
Support Centers,

A new SRM Client package is available with an update to version 1.25.

To update to the newest client package on an existing 0.4.0 or 0.4.1 OSG installation
please take the following steps:

# source $PACMAN_LOCATION/setup.sh
# cd $VDT_LOCATION
# source setup.sh
# pacman -allow save-setup
# pacman -get OSG:srm-cp-0.4.1-1

New installations of 0.4.1 will automatically pull the newest SRM Client package.

There has been a bug identified in SRM client v1.23, which
has shipped as part of OSG 0.4.1 and all VDT's since then
in integration land including vdt 1.4.0, 1.5.1.

in srm v1.25

the development vdt version 1.5.2.  We would like to see this
package made available for production OSG 0.4.1 installs as well.

When OSG 0.4.1 was deployed, this package was not part
of the VDT, it was sourced independently from the OSG pacman
site.

We are requesting that the OSG make this package available for download.
