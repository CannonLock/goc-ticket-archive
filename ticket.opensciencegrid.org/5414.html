    ORNL_NSTG Admin,

Greetings. The OSG is stopping support for the OSG 0.6.0 middleware stack and your
resource is still running that older version. The new OSG 1.0.0 middleware stack was
released in June, and support for the 0.6.0 version will stop on September 19.

respond to this ticket and let us know when you plan on doing an upgrade. If you can
not make the September 19 deadline, but would like to remain in the OSG, please let
us know so we can make arrangements. Resources for the OSG install can be found at
https://twiki.grid.iu.edu/bin/view/ReleaseDocumentation/WebHome.
