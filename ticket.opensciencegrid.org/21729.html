    Hi Iwona,

Just a couple quick things I noticed from the profile log --

Just about all the globus-* packages are not the ones shipped by OSG.  (Maybe they are the EPEL versions?)  Same for voms/voms-api-java.  Not sure if it's a problem, but I know the osg versions contain patches for several of these packages.

The other thing I've seen problems with before is when mixed versions of java are installed.  I see 'jdk' is installed as well as java-openjdk-1.6.0 and java-openjdk-1.7.0.  Sometimes it's enough to remove the 'jdk' package and then run

alternatives --config java
alternatives --config javac

and pick the java-openjdk-1.7.0 option for each.

... But it's possible that there's a different issue too...
