> You need to have the current doegrids certificate and it private key, in order to use the 'Replacement Certificate' interface to get a new

>
> If you are a globus user, then you can find the certificate and its private key in your home folder under .globus directory.

> - openssl pkcs12 -export -inkey userkey.pem -in usercert.pem -out mydoegrids.p12 -name "mydoegrids"
