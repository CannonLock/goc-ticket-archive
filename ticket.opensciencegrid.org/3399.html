    Apr 20, 2007 04:12 PM UTC by Ted Hesselroth 
    If you mean to advertise the fact that gridftpdoor part of dCache, no,
because it's describing a protocol. But, for the record, to actually
make use of the glue schema to use srm and a gridftp door, there are the
following algorithms.

srmcp -srm_protocol_version=$srmver
srm://$(GlueSEControlProtocol/GlueSEControlProtocolEndpoint)/$(GlueSA/GlueSAPath)/$vo_specific_path/filename
file:////dev/null

globus-url-copy
$(GlueSEAccessProtocol/GlueSEAccessProtocolType)://$(GlueSE/GlueSEUniqueID):$(GlueSE/GlueSEPort)/$vo_specific_path/filename
file:////dev/null

where the variables are defined as below (see
https://twiki.grid.iu.edu/twiki/bin/view/Main/GIPAttributes, but bear in
mind that is a provisional document).

A) one GlueSE for each srm door and for each gsiftpdoor
GlueSE/GlueSEUniqueID=$se_fqdn
GlueSE/GlueSEName=$se_registered_name:$type (type=disk for gsiftp,
type=srm for srm)
GlueSE/GlueSEPort=$port (port=2811 for gsiftp, port=8443 for srm)

B) one GlueSEAccessProtocol for each ftp protocol
GlueSEAccessProtocol/GlueSEAccessProtocolLocalID=gsiftp
GlueSEAccessProtocol/GlueSEAccessProtocolType=gsiftp
GlueSEAccessProtocol/GlueSEAccessProtocolEndpoint=
GlueSEAccessProtocol/GlueSEAccessProtocolVersion=
GlueSEAccessProtocol/GlueSEAccessProtocolPort=2811
GlueSEAccessProtocol/GlueSEAccessProtocolSupportedSecurity=gsi
GlueSEAccessProtocol/ GlueChunkKey=$(value of GlueSEUniqueID for the
gsiftp door)

C) one GlueSEControlProtocol for each srm protocol
GlueSEControlProtocol/GlueSEControlProtocolLocalID=srm_v$srmver

    (srmver=1 for srm v1, srmver=2 for srm v2.2)
GlueSEControlProtocol/GlueSEControlProtocolType=srm_v$srmver (srmver=1
for srm v1, srmver=2 for srm v2.2)
GlueSEControlProtocol/GlueSEControlProtocolEndpoint=httpg://$se_fqdn:8443/srm/managerv$srmver?SFN=
GlueSEAccessProtocol/ GlueChunkKey=$(value of GlueSEUniqueID for the srm
door)

D) one GlueSA for each group of vo's that can write the the same area
GlueSA/GlueSALocalID=$voname
GlueSA/GlueSARoot=$voname:$vo_specific_path
GlueSA/GlueSAPath=/pnfs/$se_domain
GlueSA/GlueSAType=permanent
GlueSA/GlueSAAccessControlBaseRule=$voname1:$voname2:$voname3
