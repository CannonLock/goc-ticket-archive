Hi All,

Actually, this is not directly a DAS problem. AFAIS, dataset presence is calculated as follows
files registered to the site in TMDB /  files in the dataset in DBS regardless of valid/invalid status

That means if a file is invalidated in a dataset, this dataset presence cannot be 100% at any site

Dataset presence: 94.56%   =  files at the site:226[1] / files in the dataset:239[2]
Since 13 files have invalid status[3], dataset presence cannot be more than 94.56%.
