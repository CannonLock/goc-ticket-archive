As for the Geant4 VO requirements, the home area for the geant4
account is visible from all worker nodes. It is up to the VO to
define any VO-specific variables in OSG, this is not done on an OSG-
wide basis on the CEs.  The home area in the case of antaeus should
be acceptable for staging software.  Note that any large files for
input and output should be staged through our OSG SE,
sigmorgh.hpcc.ttu.edu, and not through the head node of the cluster.
An appropriate storage area visible to the worker nodes of the CE
that can be accessed through the SE would be

/mnt/hep/osg/geant4


    Additional storage can be defined upon request.

Please let me know if you require any further assistance.

Alan
