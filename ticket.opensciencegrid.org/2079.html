    The following email was sent to the Support Centers and added to the RSS feed 6/12...closing ticket.

A critical patch for GIP (the Generic Information Provider) is now available from the OSG-GOC. This
fixes a reporting problem for the PBS Job Manager to the GIP. Only install this patch if you are using
PBS as the job manager and OSG version 0.4.1.

To get this critical package:

# cd $VDT_LOCATION
# source setup.sh
# pacman -get OSG:gip-patch
# $VDT_LOCATION/post-install/gris stop
# $VDT_LOCATION/post-install/gris start

Please contact the GOC with any questions.
