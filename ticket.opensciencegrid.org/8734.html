    Hi Dave,
The warning is printing because you are missing the gratia_probes entry
in the [RSV] section of config.ini.  Here's the sample:

; This setting specifies which rsv gratia probes should be used, set this to
; UNAVAILABLE to disable gratia probes, otherwise give a list of probes
separated by
; commas. Valid probes include metric, condor, pbs, lsf, sge, managedfork,
; hadoop-transfer, and gridftp-transfer
;
; On a CE, you will probably want the metric probe (for RSV), the probe
for your
; jobmanager and managedfork if you are using that
;gratia_probes = metric

Still, the "uninitialized value" warning shouldn't print (it should
provide a better message for you) and I'll improve that in an upcoming
update.
