    May 19, 2016 07:45 PM UTC by OSG-GOC 
    The error "CGSI-gSOAP running on fts00.grid.hep.ph.ic.ac.uk reports could not open connection" is similar to what happens when sites are running an FTS version that is too old or not supported by Phedex.  I suspect they are running an old version of the glite packages that use the deprecated MD5 algorithm which will not connect to modern versions of Bestman.  Or they could be running FTS3 on a new host, which is not supported by Phedex yet.  Please check the version of the glite tools from the sites that cannot connect.  The current working version that our site and other sites are running is glite-fts-client-3.7.4-9.  Anything older than this is likely to have problems like this error message.
