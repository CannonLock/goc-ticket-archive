    Hi Malina,
I don't know if your home directory modifications are causing this
problem.  RSV is failing to submit files to Condor-Cron because the
Condor submit files cannot be read.  These are dynamic files that
should be generated in:
VDT_LOCATION/osg-rsv/submissions

Can your RSV user read/write into that directory?  It should be owned
by rsv and probably 0755 permissions.

Assuming that directory looks ok you can try the following to get more
information:

# Make sure that RSV is off
rsv-control --off
# Then start it with verbose output
rsv-control --on -v3

Thanks,
scot
