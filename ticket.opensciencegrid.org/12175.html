    Jun 18, 2012 08:09 PM UTC by Alain Roy 
    Sorry, we somehow missed this ticket, my apologies!

You're right: as of right now, we don't have support for Debian in the OSG Software Stack. We've had surprisingly little negative feedback about this, so we haven't yet scheduled a plan for Debian support. My goal is to add support for Debian at some point, because I recognize there is a community that needs it, but it's not yet scheduled. If this is an urgent problem, we should have a phone discussion and figure out what reasonable subset to support on what timeframe. But we also need to be realistic about this: it's not a quick fix but will take some time to get in place.

In the meantime: we do have a subset of the OSG software stack built specifically for LIGO. It does not include the entire worker node client, but it include Globus (i.e. grid-proxy-init) but not voms-proxy-init. It won't really solve your problem, but it will help a tiny bit.

-alain
