    Feb 26, 2015 04:01 AM UTC 
    Hi Kyle,

curl uses NSS, which doesn't understand how to parse namespace files in the first place...

The basic problem is CMS is using the worker node's curl to do grid proxy work; a known fragile combination that frequently breaks in RHEL updates.  These tests should utilize the CMS-packaged version of curl from CVMFS instead.

I think we ought to kick it off to ggus and let the CMS folk deal with it.

Brian
