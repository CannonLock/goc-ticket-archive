I think I see the confusion. The grid-mapfile-local file is normally in

$VDT_LOCATION/edg/etc/grid-mapfile-local

It's normally not in /etc/grid-security unless you've done some funky changes to the configuration.

When you run edg-mkgridmap, it should merge that file (from $VDT_LOCATION/...) into the full grid-mapfile, and you should see both Marco and the RSV user in /etc/grid-security/grid-mapfile. Do you see them in that file?
