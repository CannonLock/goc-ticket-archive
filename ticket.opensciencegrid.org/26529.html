    Hi Sergeui,

Can you run osg-profile and attach the output and also rsv-control --profile.

Another work around would be to set a env section:

/etc/rsv/metrics/org.osg.srm.srmcp-readwrite.conf

[org.osg.general.vdt-version env]

X509_CERT_DIR= SET | /cvmfs/oasis.opensciencegrid.org/mis/osg-wn-client/3.2/3.2.25/el5-x86_64/etc/grid-security/certificates
X509_VOMS_DIR= SET | /cvmfs/oasis.opensciencegrid.org/mis/osg-wn-client/3.2/3.2.25/el5-x86_64/etc/grid-security/vomsdir

And restart rsv.
