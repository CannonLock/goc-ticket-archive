Problem was the LFC ACL.  From Hiro's email:

Hello.

Here is the wrong ACL.

lfc-getacl /grid/atlas/dq2/data08_cos/ESD # file:
/grid/atlas/dq2/data08_cos/ESD # owner:
/DC=org/DC=doegrids/OU=People/CN=Shawn McKee 83467 # group:
atlas/Role=production user::rwx
group::rwx        #effective:rwx
other::r-x
default:user::rwx
default:group::rwx
default:other::r-x

If you look at the parent of this directory

lfc-getacl /grid/atlas/dq2/data08_cos/
# file: /grid/atlas/dq2/data08_cos/
# owner: /DC=org/DC=doegrids/OU=People/CN=Hironori Ito 564424 # group:
atlas/Role=production user::rwx
group::rwx        #effective:rwx
group:atlas/Role=production:rwx        #effective:rwx
group:atlas/usatlas/Role=production:rwx        #effective:rwx
mask::rwx

    other::r-x
default:user::rwx
default:group::rwx
default:group:atlas/Role=production:rwx
default:group:atlas/usatlas/Role=production:rwx
default:mask::rwx
default:other::r-x

So,
You need to add

group:atlas/usatlas/Role=production:rwx
mask::rwx
default:group:atlas/Role=production:rwx
default:group:atlas/usatlas/Role=production:rwx
default:mask::rwx

Hiro

This was done and this issue is resolved.
