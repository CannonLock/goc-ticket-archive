    Apr 25, 2007 02:09 PM UTC by Philippe Canal 
    Hi,

It turns out that you ProbeConfig is miconfigured.  You need to update and
reload the backlog of activities.  Here is the procedure to follow:

In the file /usr/osg0.6.0/gratia/probe/pbs-lsf/ProbeConfig
you need to set SOAPHost to gratia-osg.fnal.gov:8880,
deactivate the cron entry and then execute this script before re-starting
the cron.

----------------------------------------------------------------------------
----
#!/bin/bash

find /usr/osg0.6.0/gratia/var/tmp/gratiafiles -name
'*_localhost_8443.gratia.xml' | while read file; do
mv -v ${file} ${file/_localhost_8443/_gratia_osg_fnal_gov_8880}
done

----------------------------------------------------------------------------
----

Cheers,
Philippe
