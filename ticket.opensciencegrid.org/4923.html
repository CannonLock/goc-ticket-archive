    these were the original instructions:

The TWiki certificate is valid, but this error will occur until you instruct your
browser to trust the ESnet Root CA. This is how you would instruct IE 7 to do this:

go to this website:
http://www.es.net/CA/ESnet-Root-CA-1/

download the DER encoded certificate to a directory where you can find it.

Go to internet options (from the browser tools -> internet options)
Go to the content tab.
Click certificates.
Display the "Trusted Root Certification Authorities."
Click import and browse to the location of the certificate.
Finish the import.

This should keep you from getting the error message. Alternatively, you can click
continue to this website, but you will have to do it each time unless you trust the
CA. If you are using a mac, you will need to import the certificate to your keychain.
If you need further instructions please notify me by responding to this ticket.

Thank you,
Elizabeth
