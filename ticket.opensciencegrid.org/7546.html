    Hi Paul,
The VDT does not directly support Ubuntu, but we do support Debian 4 and
5.  You may be able to use those Globus binaries.

I noticed that you are using this command to install:
pacman -get http://vdt.cs.wisc.edu/vdt_cache:;Globus

That command is installing VDT 1.10.1, which is not our latest release.
Maybe we need to update documentation somewhere, so please let us know
where you got the command from.  If you want the latest release - VDT
2.0.0 - you can use the following command:
pacman -get http://vdt.cs.wisc.edu/vdt_200_cache:;Globus

Note: we added support for Debian 5 in VDT 2.0.0.  I'm not sure what
Ubuntu 8.04 maps to in the Debian release world (4 or 5), but you can
probably check in /etc/debian_version?

We have more information about installing on an unsupported platform here:

Previous VDT release which you were installing from:
http://vdt.cs.wisc.edu/releases/1.10.1/installation_advanced.html#unsup_plat

Latest VDT release:
http://vdt.cs.wisc.edu/releases/2.0.0/installation_advanced.html#unsup_plat
