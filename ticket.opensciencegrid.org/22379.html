    Hi Gerard,

Digging through several layers of java files... It looks like you have to configure "voms.hostname" to be whatever you want to appear in the email.  The default is "localhost".

---
/**
* VOMS Admin service hostname. This property is needed to properly compute the service
* address when sending out email notifications.
*/
public static final String VOMS_SERVICE_HOSTNAME = "voms.hostname";
---

Can you check and see if voms.hostname is set to anything in /etc/voms-admin/lbne/voms.service.properties ?
