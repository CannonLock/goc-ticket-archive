> Can you remind me how we handle an expired OSG/DigiCert certificate? Can we re-use the same procedure here?
Usually, OIM doesn't allow user from submitting a user certificate request using already registered email address, but if the existing email address is associated with an expired DN, then OIM will allow the submission. This allows user with expired *Digicert* certificate to still make request. for DOEGrid cert, OIM doesn't know when the certificate is expired, so this logic won't apply. Again, the procedure in this case should be to remove all DN associated with the user contact.
