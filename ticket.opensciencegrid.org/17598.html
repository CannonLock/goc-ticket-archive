    exactly what steps did you do to install the CA certs, CRL for OSG.  here is what I did yesterday
on my desktop (following the instructions that I sent  - note careful reading is required because
yum install osg-ca-certs  - conflicts with yum install osg-ca-scripts )

yum install osg-ca-certs
yum install fetch-crl
yum install osg-ca-certs-updater
service osg-ca-certs-updater-cron start
chkconfig osg-ca-certs-updater-cron on
/sbin/service fetch-crl-cron start
/sbin/chkconfig fetch-crl-cron on

Then as root on can test the system -

openssl s_client -showcerts -cert /etc/grid-security/hostcert.pem -key /etc/grid-security/hostkey.pem -CApath /etc/grid-security/certificates/ -connect osg-gk.mwt2.org:2119
