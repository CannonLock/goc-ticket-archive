Planned Production BDII Switch (with short outage):
--------------------------------------------------

The GOC plans to switch the production BDII (is.grid.iu.edu: ports 2170
and 2180) to a new updated setup on Monday, October 13 2008. We expect
a short outage, likely about an hour, while we make the switch starting at
7 AM EDT (11 AM UTC).

What's new?
-----------

The new BDII setup runs the latest stable version of the CEMon Consumer,
and the latest BDII package. It also uses a simplified translation
infrastructure that converts the raw LDIF to GLUE information that BDII
can accept. It has been tested on a development server, and then at the
ITB level.

Contingency Plan:
----------------

Given that BDII is a very important tool for OSG users, the above
mentioned switch provides GOC staff the capability to roll back to the
existing setup in case of any problems with the new setup.

Future BDII Plans:
-----------------

The GOC also expects to have an additional BDII server that will be
accessible via DNS round-robin in the near future. For more information,
see https://twiki.grid.iu.edu/bin/view/Operations/BdiiPlan or contact
goc [at] opensciencegrid [dot] org.


    Contact the GOC if you have any questions.

Thanks,
Arvind (on behalf of GOC)
