    Feb 19, 2010 06:14 PM UTC by Craig Prescott 
    Hi Ian;

The full path to the system curl at UFlorida-HPC is /usr/bin/curl.

But like I said, you shouldn't have to do anything special to get a more
recent curl at UFlorida-HPC, as I've replaced the old one that come with
the VDT - just use 'curl'.

If you need a more general solution for multiple sites, you might
install your own curl that has the features you need into $OSG_APP, and
put that version at the front of your PATH.

My guess is that if you stick /usr/bin in the front of your PATH
everywhere, you might run into some trouble using tools from the VDT.

Cheers,
Craig
