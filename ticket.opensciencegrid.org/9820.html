    Derek,

Based on the OSG Blueprint, the Support center is determined at the Site level, not at the resource level.  If you change the support center for the current site that prairiefire uses, you will change it for all other resources in that site.

According to the definition:

* The OSG Blueprint Document defines a site as "A named collection of Services, Providers and Resources for administrative purposes."
* OIM uses the same definition (referred to as "Administrative Sites")
* Examples: UCSD CMS T2, ATLAS SWT2 OU, STAR Brookhaven

A site is not something physical, it's a logical named collection of services.  You are certainly in the same physical location (facility) as Nebraska, but I don't think you should be in the same site as them, with regards to the OIM topology.  Creating a new site is honestly the best option for you.

-Kyle
