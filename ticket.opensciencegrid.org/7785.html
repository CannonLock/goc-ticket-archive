Anthony Tirdani and Burt Holzman helped me clear up the issue. The
problem was that we had added a new queue 'analy_x' in PBS, but left
these values out of the config:
set queue analy_x enabled = True
set queue analy_x started = True

I added the values to the config, and Anthony sent me a patch for
pbs_common.py that adds a check for these missing values.

The classad probe now succeeds, so for me the issue is resolved.  I'm
not sure if Anthony's patch is in the main codebase yet, if not I'd like
to see it added.

- --Sarah
