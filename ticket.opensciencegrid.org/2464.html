    Hi Rick,
The certificate you got was for the host.  For site-verify and
grid-proxy-init you should really have a personal certificate.
You might make it work with using the host certificate as
a client (and you nearly did).  The private key file (hostkey.pem
or userkey.pem) needs permissions 600 so only owner can
read (or write) it.
Really best though to get a personal certificate and put it in
$HOME/.globus and leave the host certificate in /etc/grid-security.
Something like
cert-request -ou p -name "Rick Bohn" -agree -reason "put rit.edu on OSG"
etc.
Doug
