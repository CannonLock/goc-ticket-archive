    Feb 11, 2009 03:48 PM UTC by OSG-GOC 
    Hi Steve,

The central RSV collector has not received metric records for
FNAL_FERMIGRID. Please run the following profiler script, send us
that tarball (or make it accessible off the web), and then restart
RSV? Thanks!

Download Profiler:

wget http://vdt.cs.wisc.edu/scot/rsv-profiler
/bin/sh rsv-profiler
# provide access to tarball to rsv-dev by email or via web

To Restart:

cd
vdt-control --off osg-rsv condor-cron
vdt-control --on condor-cron osg-rsv

Hopefully that'll resume the RSV client. To view RSV probe and
consumer jobs ...

condor_cron_q

And you can look at:

/osg-rsv/logs/consumers/*.log
(and /osg-rsv/logs/probes/*.log)

to see there are any indications of any errors.

Let me know if you have any questions.

Thanks,
RSV Dev Team
