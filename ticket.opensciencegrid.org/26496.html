    Hi Mick,

Those are awesome news.

Regarding your problem with the gpu group in the Frontend. I find out that it goes away when the factory query expr is set to True.

So right now your config looks like this:

<match match_expr="True" start_expr="TARGET.RequiresWholeMachine">
<factory query_expr='(stringListMember("SBGridGPU", GLIDEIN_Supported_VOs))'>

I think if you exchange it to this, you might have the same result, but I have not tried it yet.

<match match_expr="'SBGridGPU' in glidein["attrs"]["GLIDEIN_Supported_VOs"]" start_expr="TARGET.RequiresWholeMachine">
<factory query_expr='TRUE'>
<match_attrs>
<match_attr name="GLIDEIN_Supported_VOs" type="string"/>
</match_attrs>
