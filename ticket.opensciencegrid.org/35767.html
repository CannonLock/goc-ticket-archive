    I am adding Petr Vokac (FZU_NOVA_CE Admin) to this ticket:

Since around the New Year there appears to have been a phase change in the number of slots available to run NOvA jobs at FZU. I've attached a Fifemon plot showing the total number of NOvA production jobs running (green) and the FZU-only running jobs (blue) to illustrate what I mean.  (I also tried taking a Grafana 'snapshot' of this plot -- not sure if that works, but the URL is https://fifemon.fnal.gov/monitor/dashboard/snapshot/UJBQL0DOJgRtxGwWCL3CtK7ZlynWSwkr.)

This could be the symptom of some maintenance, but it's lasted long enough that I'm beginning to wonder if there isn't some other reason these slots aren't being assigned.  I believe FZU is a NOvA-only site, so I don't think other experiments' jobs ought to be interfering with NOvA getting slots there.
