According to Tanya Levshina:

The bug "retrieval of checksum for srm push: convert type to lower
case, needed for ADLER32 type" was fixed on
6/19/2008 and should be available since 1.8.0.15p7 release. The latest
dcache release is
1.8.0.15p11 could be download from
http://www.dcache.org/downloads/1.8.0/index.shtml or from vdt:
http://vdt.cs.wisc.edu/software/dcache/server//2.2.8/

This upgrade needs to be done at the site which is the source for
pushmode transfers; but not by MIT.

So I think we probably should close this ticket. Thanks!
