    Richard,

What's your client version?   When did the problem start?

That error message can come as a result of many different root causes which prevent downloads from working.  The most common problem is an incorrect setting of CVMFS_HTTP_PROXY.

If you temporarily set CVMFS_DEBUGLOG=/tmp/cvmfsdebug.log in /etc/cvmfs/default.local and try mounting, you'll get more info in that file which may give you a better clue.

Dave
