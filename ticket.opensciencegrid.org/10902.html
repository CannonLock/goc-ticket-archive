    Aug 25, 2011 02:10 PM UTC by Soichi Hayashi 
    John,

As far as GOC is concerned, InteropAccounting flag is used to determine which resource will be included in our weekly "Installed Capacity Report" sent to osg-wlcg-reports@.... from oim.grid.iu.edu server.

I have following definition on the OIM / resource edit page for each flags.

x Forward BDII data to WLCG Interop BDII (for lcg-bdii.cern.ch)
x Forward RSV data to WLCG Interop Monitoring (for SAM/GridView)
x Forward Accounting Data (OIM Capacity and Benchmarking) to WLCG Interop Accounting

I am not sure if sending the "Installed Capacity Report" to osg-wlcg-report@ is part of "Forward Accounting Data to WLCG Interop Accounting", but I think we need to update our wording of InteropAccounting checkbox a little bit to make this more clear.

> The flag was set to True to allow automatic subscription to software distributions for ATLAS.
I am not sure what this means, but if this flag is used outside GOC to meann something other than what I know, please let me know!
