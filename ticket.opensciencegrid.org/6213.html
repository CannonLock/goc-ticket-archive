New Work Log Entry :
The Fermi/Remedy ticket has been resolved with the following solution.  Please resolve your ticket.

Solution : burt@.... sent this solution:

There are no enumerations of valid GlueServiceTypes in the schema and
HEPGRID_UERJ is an OSG site.  This is the fallback publication for the
authorization service on OSG when a site is using gridmap files instead
of calling out to an external service.

In short, the publication of this information is valid and should not
interfere with querying applications.  If it is, please open a new
ticket with the specifics and we will investigate.
