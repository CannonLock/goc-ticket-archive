    One more thing. One solution that was known to may fix this problem for another user was:

In the file: # this depends on your JRE version

/usr/lib/jvm/java-1.7.0-openjdk-1.7.0.79.x86_64/jre/lib/security/java.security

try setting:

jdk.tls.disabledAlgorithms=SSLv3, DHE

Java 7 hardcodes DHE keysize to 768, which Firefox now rejects.  The above will force the use of ECDHE.
