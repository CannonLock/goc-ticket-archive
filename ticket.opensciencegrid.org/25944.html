Which SE should be associated with this CE? mfosgse.ehpc.smu.edu? Or smuosgse.hpc.smu.edu? On the question of requesting resources at SMU. It is my understanding that if you use the +DESIRED_Sites="SMU_HPC" attribute in your submit file, then your job will run on any glideins available at SMU, which could be running on either smuosg1.hpc.smu.edu or mfosgce.hpc.smu.edu. If you would like to run on a specific resource at SMU, then you should use the +DESIRED_Entries="X" attribute, where X is the entry name for the resource on in the factory config. These entries names are OSG_US_SMU_smuosg1 and OSG_US_SMU_mfosgce, respectively.
