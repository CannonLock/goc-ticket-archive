Teraport is online.  Details below.

What's working:

* interactive logins
* job submissions
* Grid submissions (now includes WS-GRAM)

Almost ready:

* ATLAS DQ2 (awaiting Grid certificate)
* BIC (will contact directly; most configurations changed)
* Xen (will contact directly)

What's new:

system configuration
* RHEL4u4 installed on 130/137 nodes
- all compute, login, storage, management nodes
- tp-grid1 and tp-grid3
- RHEL4 nodes under bcfg2 control
- remainder nodes are project-specific
* all 120 computes currently online
* all nodes have current BMC, BIOS, NIC, misc. firmwares
* improved performance from remote power and console

storage configuration
* entire cluster has been upgraded to GPFS 3.1


    CI infrastructure integration
* domain migration from uchicago.edu to ci.uchicago.edu
* VLAN/subnet migration of entire cluster
* Kerberos enabled

network services
* monitoring: cacti, nagios, ganglia
- ganglia available at http://teraport.ci.uchicago.edu/monitor/ganglia/
* elimination of internal DNS and NIS
* configuration updates to DHCP servers

Grid services
* tp-grid1.ci.uchicago.edu upgraded to OSG 0.6.0
* WS-GRAM4 service now available
