    Hi Shaun,

The ManagedFork queue uses HTCondor to run jobs locally.  So, empty-condor won't suffice.  You only need HTCondor on the CE (not on the other nodes). The trick is to make sure that it will send jobs to PBS by default. I don't know whether the default is set by the first or last installed. However, the symbolic link can easily be changed after the fact. Let me know if you need more specifics.

Which document led you astray? It would be good the improve it.
