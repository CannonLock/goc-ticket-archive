This is James.  My apologies, I've been on the road in Europe and yes, we eventually figured this out.

1.  Yes, the cert had expired.
2.  We put in the new cert in /etc/grid-security/
3.  Then after some time figured out that we needed to copy the cert to /etc/grid-security/bestman/
4.  Then we discovered that the grid-mapfile had been internally munged, probably while trying to debug earlier.

We have now documented all the pieces of this configuration and NERSC should be in a much better position to support this properly.  James
