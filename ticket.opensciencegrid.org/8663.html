    Jun 2, 2010 06:56 PM UTC by Philippe Canal 
    Hi,

Your probe condor:osg-gate.rice.edu at Rice, has
not reported to the Gratia Collector since April 25th, 2010.

It is most likely due to a temporary unavailability of the collector.
The underlying problem will be fixed shortly by a new probe release.

In the meantime, you should restart the probe by killing the Gratia
probe process that are in a hung state.   If you are running Condor.
you can do this by doing:

ps -ef | grep tmp/py.in

which should return something like:

root      X  Y  0 Apr25 ?        00:00:00 ... /tmp/py.in ...

then do

kill Y X

For a pbs/lsf probe, kill the process shown by (if any)
ps -ef | grep urCollector.pl
ps -ef | grep pbs-lsf_meter
ps -ef | grep pbs-lsf.py

For an sge probe, kill the process shown by (if any)
ps -ef | grep sge_meter.py

Thanks.
